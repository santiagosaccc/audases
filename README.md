# PatternCAD

Aplicación CAD para diseño y escalado de patrones de ropa en 2D.

## Instalación

`powershell
pip install -r requirements.txt
python main.py`

## Características
- Dibujo vectorial: líneas, curvas, arcos.
- Escalado automático por tallas.
- Exportación a DXF-AAMA y PDF.
- Asistente de IA para ajustes de patrón.

## Compatibilidad
Windows, Linux, macOS (Python 3.9+)
